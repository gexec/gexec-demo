# Gexec: Demo

[![Discord](https://img.shields.io/discord/1335976189025849395)](https://discord.gg/Yda8rD4ZkJ)

This is just a demo repository which contains some really basic files to demo
the integration for Genexec for the execution of Ansible, Terraform, OpenTofu
and also different scripting languages.
